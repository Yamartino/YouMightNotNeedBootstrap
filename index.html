You Might Not Need Bootstrap

This site is an educational look into some of the components of bootstrap. It removes all of the extra fluff of bootstrap, only showing some of the core parts. Each components css has been shrunk down to the minimun required for browser support and gets ride of the extra features of each component that comes in bootstrap.

I want to preface this by saying I have nothing against bootstrap. Infact I <3 bootstrap and use it on mostly every site I build and could not live without it. So why does this matter? First of all, sometimes you can't always use bootstrap because your job or the project you are working on doesn't use it or because you are building something like an open source project and want to keep it lean. Second, this page is meant to be an educational reasource to expose some of the magic of bootstrap so that people looking to understand CSS better can copy and paste these small CSS snippets and tinker on them.
